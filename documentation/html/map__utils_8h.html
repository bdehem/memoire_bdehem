Obtain matches between two sets of descriptors Used when both sets come from keyframes (not from the map).

No match is considered if either observation already corresponds to a mapped landmark, in that case it must be matched with the map

Parameters
[in]descriptors1First set of descriptors
[in]descriptors2Second set of descriptors
[in]ptIDs1Point IDs of first set in the first keyframe (negative for unmatched points)
[in]ptIDs2Point IDs of first set in the second keyframe (negative for unmatched points)
[out]matching_indices_1Indices of matches in the first set
[out]matching_indices_2Indices of matches in the second set
[in]thresholdDistance threshold for a pair to be considered a match
[in]max_matchesMaximal number of matches allowed (no limit if maxmatches is negative)
