Detailed Description

This class defines an object which wraps all methods to perform the pose estimation based on info from user.

Only the z, rotX, rotY coordinates are taken from manual pose. The x, y, rotZ coordinates are taken from visual pose. A manual pose can be sent to the drone by using rostopic pub to publish a Pose3D message to manual_pose_channel
